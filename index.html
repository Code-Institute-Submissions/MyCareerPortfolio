Systems Engineering Process.

The key to this process is understanding the three key elements and their relationships: Operational System View; Functional System Views; Physical System Views.

System Models.

To help with understanding the system views and their interactions, it is useful to build system models. The System Modelling Language (SysML) is a popular approach and is essential to achieving a good model-based system engineering design approach, due to it's level of integration and is based on Unified Modelling Language (UML).

Evolution Through Feedback.

In order to evolve the system models with improving accuracy it will be necessary to collect key data from the actual physical systems that will inform how to adjust the algorithms based on this feedback of information.
